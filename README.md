# Verilog Projects on Tang Nano 9K

This repository contains my **basic FPGA and SystemVerilog projects** developed on the **Sipeed Tang Nano 9K** development board, using **Gowin Education IDE**.

The main goal of this repository is to document my **hands-on exploration of SystemVerilog using real hardware**, moving from simulation-only concepts to practical FPGA implementations.

The board used for all projects is the **Tang Nano 9K**, based on the **Gowin GW1NR-9 FPGA**, which provides **8640 LUT4 logic units, onboard 27 MHz clock, HDMI support, SPI flash, PSRAM, and programmable LEDs**, making it an excellent beginner-friendly FPGA platform. 

## 🔗 Board Information
- **Board:** Sipeed Tang Nano 9K
- **Official Wiki:** [Tang Nano 9K Official Wiki](https://wiki.sipeed.com/hardware/en/tang/Tang-Nano-9K/Nano-9K?utm_source=chatgpt.com)
- **FPGA:** Gowin GW1NR-9
- **IDE Used:** Gowin Education

## 📁 Repository Structure
Each folder in this repository represents **one individual FPGA project**.

Every project contains:
- source files
- constraints files
- supporting modules
- an **individual README**
- project-specific explanation and observations

This makes the repository easy to navigate and useful as a **learning portfolio of progressively built FPGA designs**.

---

## 🎯 Purpose of This Repository
These projects are intentionally **very basic**, and are focused on:

- understanding **SystemVerilog syntax**
- learning **RTL design methodology**
- verifying designs on **real FPGA hardware**
- experimenting with **clocking, counters, FSMs, LED matrices, UART, and display logic**
- building confidence with **hardware debugging using Gowin tools**

This repository serves as the **foundation of my FPGA and digital design journey**.
